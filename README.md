# SauceDemo Automation Testing

## Project Overview
Selenium Java POM Framework for SauceDemo E-Commerce website
End-to-End automation testing using Page Object Model.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Maven

## Test Scenarios
- Valid Login Test
- Add Product to Cart Test
- Cart Verify Test
- Checkout Test

## Project Structure
src/
├── pages/
│   ├── LoginPage.java
│   ├── ProductPage.java
│   ├── CartPage.java
│   └── CheckoutPage.java
├── tests/
│   └── SauceDemoTest.java
└── utils/
    └── DriverSetup.java

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Run SauceDemoTest.java as TestNG Test
